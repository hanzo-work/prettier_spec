# 部分规范 （持续更新）

## 命名规范

  1. 中划线

	    * classname

  2. 下划线：

	    * ref
	    * filename
	    * utils > function
	    * beans > const
	    * private variable

  3. 驼峰(camel, 小驼峰)
		
		* varable
  4. 帕斯卡(pascal, 大驼峰)

	      * 组件名称
	      * Class
  
  5. 全大写
    * const (not object)

## 命名方式

    1. 业务单元：业务单元全名英文
    2. 业务组件单元：
      * 布局定位名称
      * 业务功能名称
