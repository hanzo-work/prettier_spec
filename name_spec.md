# 部分规范 （持续更新）

## 命名规范

  1. 中划线

    * classname

  2. 下划线：

    * ref
    * filename
    * utils > function
    * beans > const
    * private var

  3. 驼峰

    1） 首字母大写
      * 组件名称
      * Class
    2) 首字母小写
      * var
  
  4. 全大写
    * const (not object)

## 命名方式

    1. 业务单元：业务单元全名英文
    2. 业务组件单元：
      * 布局定位名称
      * 业务功能名称
